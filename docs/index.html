    Add Face

    To add a face use this code :

    import base64
import requests

url = 'https://facial-api.globalvoxinc.com:8443/api/fr/v1/add_face'
header = {"Content-Type":"application/json"}

with open("Your Image Path", "rb") as f:
    img_str = base64.b64encode(f.read()).decode("utf-8")

myobj = {
            "key": "Your Project Key",
            "first_name" : "John",
            "last_name" : "Doe",
            "mapping_id": "1",
            "image_string" : img_str
        }
x = requests.post(url, json = myobj, headers=header)

    var url = "https://facial-api.globalvoxinc.com:8443/api/fr/v1/add_face";

var data = JSON.stringify(
    {
        "key":"Your Project Key",
        "first_name":"John",
        "last_name":"Doe",
        "mapping_id":"1",
        "check_similarity":0.5,
        "image_string":"data:image/jpeg;base64,/9j/4AA..."
    }
);

var xhr = new XMLHttpRequest();
xhr.withCredentials = true;

xhr.addEventListener("readystatechange", function() {
  if(this.readyState === 4) {
    console.log(this.responseText);
  }
});

xhr.open("POST", url);
xhr.setRequestHeader("Content-Type", "application/json");

xhr.send(data);

    Response Example

    {
    "code": 200,
    "data": {
        "face-id": "84ecbe39-a146-46b9-ab1d-d9056639b9db"
    },
    "message": "Success",
    "status": "OK"
}

    This endpoint adds a face to a project. It detects and adds the biggest face from the image.

    Add more faces of a user from different angles to identify the user correctly from any angle accurately.

    The minimum height and width of the face should be 180px.

    Request Information

    POST - /api/fr/v1/add_face

    Headers

    Key Value
    Content-type application/json

    Requests parameters

    Field Type Description
    key string Customer project authentication
    first_name string Firstname of the user
    last_name string Lastname of the user
    mapping_id string ID to group multiple faces of a single user. I.e., Group 3 faces of one user by giving same mapping id to all those faces. Can delete update multiple users using this id.
    image_string string (base64/url) Base64/Url string of image containing face you want to register
    check_face_similarity (optional) float(0-1) Optional argument if provided the number will be used to check if a face with greater than that similarity exists or not and if it exists then it returns id of the highest similar face with code 202. If it doesn’t exist then adds the face to the database.

    This endpoint is used to search face from registered images. It detects and searches the biggest face from the image. In return it gives id, similarity, first_name, last_name, status of 3 best similar faces. If more faces of a user is registered then this can be used to show the confidence of the input image.

    Similarity – It specifies how similar the given image is from registered images. Recommended similarity is 0.6, any similarity greater than or equal to 0.5 is considered recognized face otherwise unknown face.

    The minimum height and width of the face should be 180px.

    Request Information

    POST - /api/fr/v1/search_face

    Headers

    Key Value
    Content-type application/json

    Requests parameters

    Field Type Description
    key string Customer project authentication
    image_string string (base64/url) Base64/Url string of image containing face you want to search

    Response Codes

    The API uses the following response codes:

    Response Code Description
    100 Internal error
    101 Database error
    102 Unknown error
    200 Success
    201 Indexing Error
    202 Existing face id
    600 Error adding data
    601 Error updating data
    602 Error deleting data
    603 Error getting data
    604 Parameter error
    607 Invalid ID/key
    608 Inactive status
    609 No access
    1001 Base64 Image String is not proper.
    1002 Face not found.
    1003 Face size should be greater than 179X179.
    1004 Face is not proper error generating embeddings.
    1005 Please add a face before recognition.
    1006 Failed to delete face.
